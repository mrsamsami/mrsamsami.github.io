Physical reasoning with reinforcement learning

We try to develop agents that understand and reason about physics and simple classical mechanics in 2D environments. Specifically, by using causal inference tools and reinforcement learning, we try to propose methods to learn representations capturing the invariant properties of a particular set of environments.
